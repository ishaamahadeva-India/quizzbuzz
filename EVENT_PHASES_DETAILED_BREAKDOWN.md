# Detailed Event Phases Breakdown: T20, ODI, Test Matches

## Overview

This document provides a **comprehensive breakdown** of events for each phase of T20, ODI, and Test matches. Each event includes:
- **Unlock Timing**: When the event becomes available
- **Lock Timing**: When predictions close
- **Points**: Suggested point value
- **Difficulty**: Easy, Medium, Hard

---

## ODI Match Event Phases

### Key Differences from T20:
- **50 overs per innings** (vs 20 in T20)
- **Two powerplays**: Overs 1-10 and Overs 31-40
- **Longer middle overs**: Overs 11-30
- **More strategic events**: 300+, 400+ scores

### PHASE 1: Pre-Match (ODI)

Same as T20, plus:
- 300+ Score Prediction
- 400+ Score Prediction
- Chase Successful Prediction

**T20/IPL**:
- DRS Reviews Per Innings (0-1, 2-3, 4+)
- DRS Review in Powerplay (Yes/No)
- DRS Review in Death Overs (Yes/No)

**ODI**:
- DRS Reviews Per Innings (0-2, 3-4, 5+)
- DRS Review in Powerplay 1 (Yes/No)
- DRS Review in Powerplay 2 (Yes/No)

**Test**:
- DRS Reviews Per Day (0-2, 3-4, 5+)
- DRS Reviews Per Session (0-1, 2-3, 4+)
- Most DRS Reviews in an Innings (0-3, 4-6, 7+)
